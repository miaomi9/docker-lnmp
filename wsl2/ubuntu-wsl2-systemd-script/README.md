## 不进入 `systemd`

> 执行 `wsl -d wsl-k8s` 不进入 `systemd`

新建 `/non-systemd` 文件

**Windows**

```powershell
$ New-Item \\wsl$\wsl-k8s\non-systemd

# 移除
# $ Remove-Item \\wsl$\wsl-k8s\non-systemd
```
